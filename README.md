This project implements an Elephant Detection System using the YOLO (You Only Look Once) object detection algorithm. It is designed to identify elephants in images or video streams in real-time, which can be useful for wildlife monitoring, conservation efforts, and preventing human-elephant conflicts.



🚀 Features
Real-time elephant detection using YOLO models
Supports images, videos, and live webcam feed
High accuracy and fast inference
Easy-to-use interface for testing
Custom-trained model support




🧠 Model Used
 YOLOv8 (you can specify your version)
Pretrained weights fine-tuned on elephant dataset
